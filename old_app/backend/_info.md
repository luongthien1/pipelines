# Backend Module — FastAPI & YOLO Training

This directory contains the FastAPI server, database logic, and model training services.

| Name | Type | Description |
|------|------|-------------|
| `models/` | Directory | SQLAlchemy & Pydantic models for the system. |
| `routers/` | Directory | API endpoint definitions (Projects, Datasets, Models). |
| `services/` | Directory | Business logic: Training (YOLO26s), Image Processing, etc. |
| `main.py` | File | Application entry point and configuration. |
| `db.py` | File | Database connection and session management. |
| `_info.md` | File | This file. Notes children's purpose. |
